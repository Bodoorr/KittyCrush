### **_Description_**

KittyCrush is a fun interactive Valentine web app that reacts to your facial expressions in real time.  
When the user smiles, hearts rain down and a cute Valentine message appears. If they frown, the app playfully tries to make them smile with a funny filter effect.

The app uses live camera detection to recognize emotions instantly and create a playful experience.  
It combines animation, real-time face tracking, and image capture to generate a shareable Valentine snapshot.

## **_Special Features_**

1. **Real-Time Smile Detection**  
   Detects user expressions instantly using the camera.

2. **Heart Rain Animation**  
   Cute animated hearts appear when a smile is detected.

3. **Auto Snapshot Capture**  
   Takes a screenshot automatically when the user smiles.

4. **Cloud Upload**  
   Saves captured images to the cloud.


5.  **Mobile Friendly**  
   Designed to work smoothly on mobile devices.
